Bot Manager¶

class bot_manager.BotManager(bot_list: dict = None, next_index: numpy.uint8 = 0, bot_statuses: numpy.ndarray = None)¶

Manages the system of Bots and keeps track of where they are on the field.

Keeps track of and assigns threads for bots to take action on. (Aka their engine).

Parameters

  • bot_list (Array of bot_skeleton.Bot) – Array of bots currently being managed. Defaults to None.

  • next_index (uint) – Keeps track of the next available integer to assign as a bot id. Defaults to 0.

  • bot_statuses (Array of uint8) – Bit array representing each bot’s status in the timestep. 1 for complete, 0 for incomplete. Index of number is bot’s id.

activate_si()¶

Sequential bot activations. Bot activations are arbitrary as long as each one gets equal time to run during each time step, so they just run in the order that they were created.

Initializes self.bot_statuses.

Once all bot_statuses are 1, this method returns 1.

Returns

uint 0|1
